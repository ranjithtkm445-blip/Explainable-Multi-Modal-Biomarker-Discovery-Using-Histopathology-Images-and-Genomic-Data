# Explainable-Multi-Modal-Biomarker-Discovery-Using-Histopathology-Images-and-Genomic-Data
AI-powered system combining histopathology image analysis (EfficientNet-B0 + Grad-CAM) with genomic risk prediction (Random Forest + SHAP) across 100 cancer genes. Predicts tumor presence and identifies key biomarkers with full explainability. Built on PatchCamelyon dataset with 50 patient profiles.
